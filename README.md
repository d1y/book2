## setup

```console

# 本地`clone`起来
git clone https://github.com/d1y/book2

# 下载静态资源
chmod u+x startup.sh
./startup.sh

# 安装依赖
yarn

# 热启动服务
yarn serve

# 编译
yarn run build

# 运行测试
yarn test

# Lints and fixes files
yarn lint

```
